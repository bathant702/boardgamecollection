About Me

Thank you for checking out this app. This app was developed with a combination of:

  • • HTML
  • • Materialize CSS
  • • Python/Django
  • • Deployed with Heroku
While this app is far from finished, I hope you enjoyed the base functionality it can provide a user in the board game hobby. Come back often to see the development and implementation of the features mentioned:
  • • Most Popular Genre of the Month, followed by a list of games relating to that genre.
  • • Picture upload for users.
  • • Game ratings.
  • • Linking to external information and resources for the user game.
  • • Where to buy search function.
